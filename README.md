# @geolonia/detic

A simple reverse geocoder library.

## install and usage

```shell
$ npm install @geolonia/detic
$ detic jp 0,0                  // false
$ detic jp 139.925237,35.546346 // { "prefCode": "22" }
```

## Development

```shell
$ git clone git@github.com:geolonia/detic.git
$ cd detic
$ npm install # or yarn
$ npm test
```
